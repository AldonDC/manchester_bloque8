# 🤖 TE3003B: Integration of Robotics and Intelligent Systems 2026

Welcome to our project repository for the **TE3003B** block. This repository serves as our centralized development space for all robotics challenges and the final project (The Challenge).

## 📁 Repository Structure

*   **/challenges**: Solutions to weekly robotics challenges.
*   **/reto**: Our final integrated robotics/AI project (Coming Soon 🚀).
*   **/professor_repo**: A copy of the [Manchester Robotics](https://github.com/ManchesterRoboticsLtd/TE3003B_Integration_of_Robotics_and_Intelligent_Systems_2026) base repository for easy reference.
*   **/.github/workflows**: Automated CI/CD pipelines to ensure our code works on Ubuntu with ROS2 Humble.

## 🛠️ Tech Stack

*   **OS**: Ubuntu 22.04 LTS (Jammy)
*   **ROS2 Version**: Humble Hawksbill
*   **Programming Languages**: Python 3.10+, C++
*   **Tools**: Git Actions, VS Code

## 🚀 CI/CD Pipeline

We've integrated GitHub Actions to automatically:
1.  Check Python linting and formatting.
2.  Validate ROS2 package dependencies.
3.  Simulate/Build our workspace in a ROS2 Humble environment.
